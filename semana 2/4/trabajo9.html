¿Que son los formularios en Html y para que se usan?

son secciones de un sitio web que contiene elementos interactivos como campos de texto, casillas de verificacion y botones, para permitir al usuario ingresar y enviar datos a un servidor web ,esto se utiliza para poder recopilar informacion del usuario.

¿Que son las alertas en JavaScript?

Son cuadros de dialogo que aparecen en el navegador para mostrar un mensaje al usuario utilizando la funcion alert(), lo que significa que el codigo posterios no se ejecuta hasta que se descarte la alerta.

La funcion onclick

Es un evento que se ejecuta cuando un usuario hace clic en un elemento HTML como un boton o un enlace, se puede usar para activar acciones como mostrar un mensaje de alerta o dirigir a otra pagina.
