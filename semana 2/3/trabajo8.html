¿Que es AppendChild?

Es un metodo en JavaScript que se utiliza para agregar un nuevo nodo como el ultimo hijo de un nodo padre existente. Tambien es una funcion fundamental par amanipular el DOM,permitiendo añadir elementos dinamicamente a una paguina web, si el nodo que se esta agregando ya esta en el documento, AppendChild lo mueve de su posicion actual a la nueva ubicacion.
