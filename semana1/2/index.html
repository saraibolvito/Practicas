¿cual es la estructura o modelo de Flexbox y Grid?

Los dos son modelos de diseño de CSS para crear interfaces web adaptables, Flexbox es ideal para diseños unidimensionales como en filas o una columna. Grid esta diseñado para diseños bidimensionales como en filas y columnas simultaneamente.

¿Que es un pseudo elemento en CSS?

Un pseudo-elemento es el que permite aplicar estilos a partes especificas de un elemento HTML sin modificarlo el HTML directamente

  • :: before: insetra contenidos antes del contenido real del elemento
  • :: after: insertar contenidos despues del contenido real
  • :: first-letter: estiliza la primera letra de un bloque de texto
  • :: first-line: estiliza la primera linea de un bloque de texto
  • :: placeholder: estiliza el texto del atributo placeholder en inputs
  • :: selection: estiliza el texto seleccionado por el usuario

nth-child

permite seleccionar elementos segun su pocision dentro de su elemento padre,la (n) puede ser un numero o una formula o una palabra clave

hover

se utiliza para aplicar estilos cuando el usuario pasa el mouse sobre el elemento como efectos interactivos,como cambiar colores,mostrar menus o animaciones
