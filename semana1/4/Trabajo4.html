¿Que es Java script?

Es un lenguaje de programacion que los desarrolladores utilizan para hacer paguinas web interactivas. Desde actualizar fuentes de redes sociales a mostrar animaciones y mapas interactivos.

¿Que es var?

Es una palabra clave para la declaracion de variables que permiten al compilador o interprete inferir el tipo de datos de la variable. El proposito es indicar que el texto entre la etiqueta es un identificador o nombre de la variable.

¿Que es let?

Es una palabra clave para declarar variables que tienen alcance de bloque y pueden ser reasignadas.

¿Que es const?

Se usa para declarar variables cuyo valores no pueden ser reasignados despues de su inicializacion, y su alcance es de bloque.

¿Que son las funciones y los if?

Las funciones son un fragmento de codigo que realiza una tarea especifica y puede ser llamado multiples veces como proposito o estructura

Los if se utilizan para controlar el flujo del programa y ejecutar codigo de manera condicional.
