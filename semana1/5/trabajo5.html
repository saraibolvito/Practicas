多Que son los ciclos en JavaScrip

Tambien es llamado bucles, son estructuras de control que permiten ejecutar un bloque de codigo repetidamente mientras se cumpla una condicion, o un numero especifico de veces.

多Que es for?

Es una estructura de control que permite ejecutar un bloque de codigo un numero especifico de veces.

多Que es while?

Es un tipo de bucle que repite un bloque de codigo miestras una condicion especifica sea verdadera.

多Como se hace un for?

Se debe usar la palabra for seguida de parenteis que contiene partes que esta la inicializacion de un avariable (x = 0 ), una condicion para que el bucle continue ( x < 10),el siguente paso es actualizar la variable ( x++), dentro de las llaves{} se coloca el codigo que se repetira en cada iteracion.

for ( let i = 1; i < 10; i++) { console.log(i); }
